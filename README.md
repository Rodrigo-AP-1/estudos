# ESTUDO

Anotações e exercícios de cursos que faço por fora da faculdade.

## O que tem aqui

**c-brocode** — começo de C, seguindo o curso do Bro Code. Uma pasta por aula.

**html-css-brocode** — HTML e CSS do mesmo canal. Páginas pequenas, geralmente `index.html` + `style.css`.

**html-css-cursoemvideo** — exercícios e desafios do Curso em Vídeo (Gustavo Guanabara).

## HTML

Pra ver qualquer página, abre o `.html` no navegador. Não precisa instalar nada.

Até agora mexi com links, imagens, vídeos, favicon, páginas múltiplas e um pouco de CSS (margin, padding, cores, fontes).

## C

Mesma ideia da faculdade:

```bash
gcc main.c -o main
./main
```
